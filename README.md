# R scripts for steroid data processing and analysis

## Description
This repository contains the R scripts used for data processing and statistical analysis in a study examining associations between subjective childbirth experience, maternal steroid profiles, as well as early infant outcomes, maternal-infant bonding and perinatal depression.

## Study context
The analyses were conducted as part of a larger research project funded by La Marató de TV3 (project 202204-10). The study focuses on early postpartum psychoneuroendocrine processes.

## Contents
- Data cleaning procedures
- Statistical analyses (correlation analyses and robust linear regression models)
- Multiple testing correction

## Data availability
The datasets used in this study contain sensitive clinical and biological information and are not publicly available due to ethical and privacy restrictions.

## Requirements
The scripts were developed using R. Required packages may include:
- xlsx 
- dplyr 
- stringr
- ggplot2
- kableExtra
- janitor
- car
- MASS
- sandwich
- tidyr
- robustbase
- ggpubr

## Usage
The scripts are intended to illustrate the analytical workflow and can be adapted to similar datasets. Users should modify file paths and variable names according to their data.

## Reproducibility
Due to data protection restrictions, the original dataset is not included. The code is provided to facilitate transparency and reproducibility of the analytical approach.
